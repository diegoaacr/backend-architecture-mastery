# Backend Architecture Mastery 2026

Repositorio completo para aprender **arquitectura backend desde cero hasta nivel avanzado**, con enfoque práctico, profesional y orientado a ingeniería real.

---

## 🎯 Objetivo

Aprender arquitectura de software en backend de forma clara, paso a paso, hasta poder:

* Diseñar sistemas backend escalables y mantenibles
* Entender cómo se estructuran aplicaciones reales
* Tomar decisiones arquitectónicas correctamente
* Trabajar con patrones, buenas prácticas y sistemas distribuidos
* Pensar como un backend engineer profesional

---

## 🚀 Cómo usar este repositorio

Sigue este orden:

1. `00-introduction` → Entender cómo funciona el curso
2. `01-fundamentals` → Base sólida de conceptos
3. `02-core-architectures` → Primeras arquitecturas
4. `03-design-patterns` → Herramientas de diseño
5. `04-clean-and-domain` → Arquitectura profesional
6. `05+` → Avanzado, distribuido, infraestructura, etc.

👉 No saltes pasos. Todo está conectado.

---

## 📚 Estructura del contenido

* `docs/` → Explicaciones claras por niveles

## 🧩 Tecnologías usadas

Los ejemplos estarán basados principalmente en **JavaScript / TypeScript (Node.js)**.

👉 Los conceptos son independientes del lenguaje.

## 🧠 Qué vas a aprender

* Fundamentos de backend y arquitectura
* Principios de diseño (SOLID, KISS, DRY, YAGNI)
* Paradigmas de programación
* Arquitecturas (monolito, capas, modular, etc.)
* Design Patterns
* Clean Architecture y DDD
* Arquitecturas avanzadas (CQRS, Event Sourcing, etc.)
* Sistemas distribuidos
* Comunicación (REST, GraphQL, eventos)
* Bases de datos y persistencia
* Seguridad en backend
* Deploy, infraestructura y CI/CD
* Observabilidad y calidad
* Decisiones arquitectónicas reales
