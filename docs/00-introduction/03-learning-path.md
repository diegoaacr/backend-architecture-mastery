# Learning path

Ruta general para aprender arquitectura backend de forma progresiva.

---

## 🎯 Objetivo

Tener una visión clara del camino de aprendizaje antes de empezar.

---

## 🧠 Etapas del aprendizaje

### 1. Fundamentos

Aprendes las bases del backend y arquitectura.

* Qué es backend
* Cómo funciona una aplicación
* Conceptos clave

---

### 2. Arquitecturas base

Aprendes a estructurar proyectos simples.

* Arquitectura por capas
* MVC
* Monolitos

---

### 3. Design Patterns

Aprendes herramientas para resolver problemas comunes.

* Patrones creacionales
* Patrones estructurales
* Patrones de comportamiento

---

### 4. Arquitectura profesional

Aprendes a separar correctamente un sistema.

* Clean Architecture
* DDD básico
* Casos de uso

---

### 5. Sistemas avanzados

Entiendes cómo escalan los sistemas reales.

* Microservicios
* Event-driven architecture
* Sistemas distribuidos

---

### 6. Infraestructura y producción

Aprendes cómo funcionan los sistemas en la vida real.

* Deploy
* CI/CD
* Cloud

---

### 7. Decisiones arquitectónicas

Aprendes a pensar como ingeniero.

* Cuándo usar cada arquitectura
* Trade-offs
* Escalabilidad

---

## ⚠️ Importante

No necesitas dominar todo de inmediato.

👉 Avanza paso a paso.
