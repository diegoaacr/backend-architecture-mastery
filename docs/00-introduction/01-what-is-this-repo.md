# What is this repository

Este repositorio está diseñado como un curso completo de arquitectura backend.

---

## 🎯 Objetivo

Que puedas aprender arquitectura backend desde cero hasta nivel profesional, de forma clara, paso a paso y sin confusión.

---

## 📚 Cómo recorrer el contenido

Sigue siempre el orden de las carpetas:

1. `00-introduction`
2. `01-fundamentals`
3. `02-core-architectures`
4. `03+` (niveles intermedios y avanzados)

👉 No saltes temas. Cada bloque se basa en el anterior.

---

## 🧠 Cómo estudiar

Para cada tema:

1. Lee la explicación
2. Entiende el concepto
3. Relaciónalo con otros temas
4. Revisa ejemplos (cuando aplique)

---

## 🧪 Practica siempre

No basta con leer.

* Diseña sistemas simples
* Piensa en cómo organizar código
* Analiza aplicaciones reales
* Aplica conceptos en proyectos

---

## 📂 Estructura del repo

* `docs/` → Explicaciones paso a paso
* `examples/` → Ejemplos simples
* `exercises/` → Práctica por nivel
* `labs/` → Ejercicios guiados
* `projects/` → Simulación de sistemas reales

---

## ⚠️ Importante

La arquitectura no se aprende memorizando.

Se aprende entendiendo cómo funcionan los sistemas.
