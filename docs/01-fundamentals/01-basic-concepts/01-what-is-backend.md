# What is backend

## 🎯 Objetivo

Entender qué es el backend y cuál es su rol dentro de una aplicación.

---

## 🧠 Explicación simple

El backend es la parte del sistema que no ves.

Es donde ocurre la lógica, se procesan los datos y se conecta con la base de datos.

---

## 🧩 Ejemplo

Cuando usas una aplicación:

1. Ingresas tus datos (por ejemplo, login)
2. Esa información se envía al backend
3. El backend valida los datos
4. Consulta la base de datos
5. Devuelve una respuesta

👉 Todo eso ocurre en el backend.

---

## 🏗️ Qué hace el backend

El backend se encarga de:

* Procesar información
* Aplicar lógica de negocio
* Conectar con bases de datos
* Manejar autenticación
* Responder a peticiones

---

## 🔗 Relación con frontend

* Frontend → lo que el usuario ve
* Backend → lo que hace que todo funcione

👉 Ambos trabajan juntos.

---

## 💡 Idea clave

El backend es el cerebro de la aplicación.

---

## ⚠️ Errores comunes

* Pensar que backend es solo base de datos
* Pensar que backend es solo APIs
* Ignorar la lógica de negocio
